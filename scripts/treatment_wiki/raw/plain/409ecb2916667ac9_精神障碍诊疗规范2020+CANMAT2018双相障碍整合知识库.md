## 一、急性躁狂的药物治疗（Section 3）

### 1.1 躁狂诊断要点（DSM-5）

- 异常持续的心境高涨/膨胀/易激惹 + 活动量/精力增加，持续≥1周（住院可<1周）
- 需≥3项B症状（如心境仅为易激惹则需≥4项）
- 需导致明显功能损害或住院或伴精神病性症状
- DSM-5取消"混合发作"，改为"伴混合特征"描述符

### 1.2 激越的阶梯式管理

**一线（First-line）：**

|------|---------|---------|---------|----------|
| 洛沙平（吸入） | 吸入 | **1** | 5 mg | 10 mg |
| 阿立哌唑 | IM | 2 | 9.75 mg | 15 mg |
| 劳拉西泮 | IM | 2 | 2 mg | — |
| 奥氮平 | IM | 2 | 2.5 mg | 10 mg |

**二线（Second-line）：**

### 1.3 急性躁狂药物治疗（5步法）

#### Step 1：评估与排查

- 评估暴力/自杀风险、自知力、共病（尤其物质使用）、支持系统
- 排除药物滥用、药物所致症状、躯体疾病
- **停用所有抗抑郁药**
- 如有既往诊断，立即开始抗躁狂治疗

#### Step 2：启动一线治疗

**一线单药治疗（约50%患者3-4周内应答）：**

| 药物 | 证据等级 | 备注 |
|------|---------|------|
| **锂盐** | **1** | 首选推荐，效力可比较（Cohen's d 0.32-0.66） |
| **喹硫平** | **1** | — |
| **丙戊酸（双丙戊酸）** | **1** | — |
| **阿塞那平** | **1** | — |
| **阿立哌唑** | **1** | — |
| **帕利哌酮** | **1** | 剂量>6 mg |
| **利培酮** | **1** | — |
| **卡利拉嗪** | **1** | — |

> 降级为二线的药物（有效但安全性/耐受性顾虑）：
> 奥氮平（1）→ 体重增加风险；卡马西平（1）→ 药物相互作用；齐拉西酮（1）→ QTc；氟哌啶醇（1）→ EPS

**一线联合治疗：**

| 联合方案 | 证据等级 | 备注 |
|---------|---------|------|
| 喹硫平+锂/丙戊酸 | 1 | — |
| 阿立哌唑+锂/丙戊酸 | 2 | — |
| 利培酮+锂/丙戊酸 | 1 | — |
| 阿塞那平+锂/丙戊酸 | 2 | — |

> 联合治疗 vs 单药：平均约多20%患者应答。对需要快速应答、既往单药部分应答、严重躁狂者推荐联合。
>
> **注意：** 不推荐帕利哌酮或齐拉西酮联合锂/丙戊酸（证据阴性）。

#### Step 3：换用/加用其他一线药物

- 先确认剂量已优化、依从性、排除物质使用
- 所有抗躁狂药1-2周内应出现分离，2周无效应考虑换药/加药
- 尝试多种一线策略后再考虑二线

| ECT | 3 | 可达80%显著改善，双额电极优于双侧 |

### 1.5 临床特征指导治疗选择

| 临床特征 | 推荐药物 |
|---------|---------|
| **经典欣快夸大型躁狂**（无抑郁症状） | **锂盐首选** |
| 少数既往发作，躁狂-抑郁-缓解交替 | **锂盐** |
| 家族史BD，尤其锂盐应答家族史 | **锂盐** |
| **多次发作、易激惹/恶劣心境** | **丙戊酸** |
| **共病物质滥用** | **丙戊酸** |
| 头外伤史 | 丙戊酸或卡马西平 |
| **混合特征** | **非典型抗精神病药 + 丙戊酸**，常需联合 |
| 焦虑症状 | 丙戊酸、喹硫平、奥氮平可能有特异性抗焦虑效应 |
| 精神病性症状（心境不一致） | 非典型抗精神病药 ± 情感稳定剂 |
| 快速循环 | 无特定药物优势，按维持期疗效选药；常需联合治疗；停用抗抑郁药/兴奋剂 |
| 季节模式 | 无特定药物优势 |

## 二、急性双相抑郁的药物治疗（Section 4）

### 2.1 诊断挑战

- BDI抑郁常被误诊为MDD（缺乏回顾性轻躁狂/躁狂筛查）
- 提示BD而非MDD的特征：早发（<25岁）、反复发作、BD家族史、精神病性抑郁、非典型特征（睡眠过多/食欲增加）、抗抑郁药诱发躁狂

### 2.2 双相抑郁药物治疗（5步法）

#### Step 1：评估

- 评估抑郁严重度、自杀风险（>70%自杀死亡发生在抑郁期）、依从性、支持系统
- 混合特征抑郁 = **极高自杀风险期**
- 排除药物/酒精/躯体因素
- 最常见自杀方式 = 自我中毒（注意药物毒性风险：锂盐致死风险低于卡马西平；阿片类和BZD最常致死）

#### Step 2：启动一线治疗

**一线单药治疗：**

| 药物 | 证据等级 | 推荐剂量 | 备注 |
|------|---------|---------|------|
| **喹硫平** | **1** | **300 mg/d**（300mg与600mg等效） | 无治疗前首选 |
| **锂盐** | **2** | 血药浓度 **0.8-1.2 mEq/L** | 已用锂盐者；低锂浓度（0.61）可能无效 |
| **拉莫三嗪** | **2** | ≥200 mg/d | 需缓慢滴定（皮疹风险），起效慢 |
| **鲁拉西酮** | **2** | — | 第1周即可分离，适合需快速应答者 |

**一线辅助治疗（加用）：**

| 药物 | 证据等级 | 适用场景 |
|------|---------|---------|
| **鲁拉西酮** | **1** | 加用于锂盐/非典型抗精神病药 |
| **拉莫三嗪** | **2** | 加用于锂盐/喹硫平 |

> **锂盐一线推荐理由说明**：唯一大型双盲安慰剂对照试验锂盐未优于安慰剂（平均血锂0.61 mEq/L偏低），但多项小型交叉试验显示锂盐有效。锂盐血浓度≥0.8 mEq/L时与帕罗西汀等效。综合急性抑郁+预防+抗躁狂+抗自杀，列为一线。
>
> **拉莫三嗪一线推荐理由说明**：4项双盲试验主要终点阴性，但荟萃分析有效。试验设计缺陷：滴定期占8周试验大部分时间、目标剂量偏低（200 mg）、达目标剂量时间短。辅助治疗（加锂/喹硫平）有阳性结果。耐受性好。

#### Step 3：换用/加用其他一线药物

- 早期改善（2周）可预测总体应答；拉莫三嗪例外（需缓慢滴定）
- 无改善 → 优化剂量、确认依从性 → 换药或加药
- 加药 vs 换药原则：综合考虑多病程目标（如锂盐用于抑郁但同时也提供抗躁狂预防 → 加药）

#### Step 4：二线药物

| 药物/方案 | 证据等级 | 备注 |
|---------|---------|------|
| 丙戊酸单药 | 2 | — |
| SSRI/安非他酮辅助治疗（+锂/丙戊酸/非典型抗精神病药） | 1（荟萃分析） | 效应量小但有效；**BDI抑郁禁用抗抑郁药单药** |
| ECT | 3 | 适合难治、快速应答需求（严重自杀、木僵、精神病性抑郁） |
| 卡利拉嗪 | 2 | 大型RCT + 汇总分析 |
| **奥氮平/氟西汀合剂（OFC）** | **2** | 第1周即分离，适合快速应答 |

> **抗抑郁药使用关键变更**：2018版CANMAT将辅助SSRI/安非他酮从一线降为二线——荟萃分析虽支持有效，但效应量小。ISBD建议：**避免在抗抑郁药诱发躁狂/轻躁狂史、当前/突出混合特征、近期快速循环患者中使用抗抑郁药。**

#### Step 5：三线药物

| 药物/方案 | 证据等级 |
|---------|---------|
| 卡马西平单药 | 2 |
| 奥氮平单药 | 1 |
| 阿立哌唑辅助 | 4 |
| 阿莫达非尼辅助 | 4 |
| 阿塞那平辅助 | 4 |
| 卡利拉嗪 → 见二线 |
| 左甲状腺素辅助 | 3 |
| 莫达非尼辅助 | 2 |
| 普拉克索辅助 | 3 |
| rTMS（左/右DLPFC） | 3 |
| SNRI/MAOI辅助 | 2（需确保充分抗躁狂预防） |
| EPA（二十碳五烯酸）辅助 | 2 |
| N-乙酰半胱氨酸辅助 | 3 |
| 光疗（含中午亮光） | 3 |
| 光疗+总睡眠剥夺辅助 | 2 |
| **氯胺酮IV** | **3** | 快速高效但效果短暂，限用于严重/难治/高自杀风险 |

### 2.3 双相抑郁不推荐的药物

| 药物 | 证据 |
|------|------|
| **抗抑郁药单药（BDI抑郁）** | **2阴性** |
| **阿立哌唑单药** | **1阴性**（MADRS差仅1.12分，无临床意义） |
| **齐拉西酮单药/辅助** | **1阴性** |
| 拉莫三嗪+叶酸 | 2阴性 |
| 米非司酮辅助 | 2阴性 |

### 2.4 临床特征指导双相抑郁治疗

| 临床特征 | 推荐药物 |
|---------|---------|
| **需快速应答** | 喹硫平（第1周分离）、鲁拉西酮（第1周）、OFC（第1周）、卡利拉嗪（第1周）；ECT可提前使用 |
| **焦虑症状** | 喹硫平（优于安慰剂缓解焦虑）、OFC；丙戊酸/利培酮/拉莫三嗪抗焦虑效应有限 |
| **混合特征** | 非典型抗精神病药（类效应）；OFC、阿塞那平、鲁拉西酮均有效；**避免抗抑郁药** |
| **精神病史** | 临床经验：ECT + 抗精神病药高度有效 |
| **快速循环** | 锂/丙戊酸/奥氮平/喹硫平维持疗效相当；**拉莫三嗪无效**；**禁用抗抑郁药** |
| **既往抗抑郁药有效（无转躁）** | 辅助抗抑郁药可能合适 |
| **忧郁特征** | 临床经验：ECT非常有效 |
| **非典型特征** | 反苯环丙胺（需+锂/丙戊酸/非典型抗精神病药） |

## 三、双相障碍维持治疗（Section 5）

### 3.1 长期治疗的必要性

- BD可能是神经进展性疾病：复发 → 脑灰/白质减少、认知恶化、发作间恢复下降
- 治疗：年复发率19-25%（治疗）vs 23-40%（安慰剂）
- 50-90%停锂者3-5月内复发；快速停药风险更高
- 首发后即应启动综合治疗（证据支持逆转认知损害、保护脑可塑性）

### 3.2 维持治疗药物阶梯

#### 一线治疗

**单药：**

| 药物 | 证据等级 | 抗躁狂预防 | 抗抑郁预防 | 备注 |
|------|---------|----------|----------|------|
| **锂盐** | **1** | ✅ 强 | ✅（弱于躁狂） | 金标准；抗自杀效应；血锂0.8-1.2 mEq/L |
| **喹硫平** | **1** | ✅ | ✅ | 混合特征者尤佳 |
| **丙戊酸** | **1** | ✅ | ✅ | 唯一大型RCT未优于安慰剂但属"失败试验"；多项支持证据 |
| **拉莫三嗪** | **1** | ❌ 弱 | ✅ 强 | 抑郁极性优势；不适合频繁躁狂发作者 |
| 阿塞那平 | 2 | ✅ | ✅ | 同时预防躁狂和抑郁 |
| 阿立哌唑口服/月剂 | 2 | ✅ | ❌（未证明） | 安全性/耐受性好 |

**联合：**

| 联合方案 | 证据等级 |
|---------|---------|
| 喹硫平+锂/丙戊酸 | 1（预防任何发作、躁狂、抑郁） |
| 阿立哌唑+锂/丙戊酸 | 2 |

> **关键建议**：急性期联合治疗有效的患者 → 前6个月继续联合治疗（2级），6个月后重新评估是否继续联合。

#### 二线治疗

| 药物/方案 | 证据等级 | 备注 |
|---------|---------|------|
| 奥氮平单药 | 1 | 代谢综合征降为二线 |
| 利培酮长效注射单药 | 1 | 预防任何/躁狂发作有效，抑郁预防不确定 |
| 利培酮长效注射辅助 | 2 | — |
| 卡马西平 | 2 | 无大型安慰剂对照试验 |
| 帕利哌酮 | 2 | 优于安慰剂但劣于奥氮平 |
| 齐拉西酮辅助 | 2 | — |
| 鲁拉西酮辅助 | 2 | 以抑郁发作者尤佳 |

#### 三线治疗

| 药物/方案 | 证据等级 |
|---------|---------|
| 阿立哌唑+拉莫三嗪 | 2（额外躁狂预防） |
| 氯氮平辅助 | 4 |
| 加巴喷丁辅助 | 4 |
| 奥氮平/氟西汀合剂 | 2 |

### 3.3 维持治疗不推荐的药物

| 药物 | 证据 |
|------|------|
| 哌泊清+情感稳定剂 | 2阴性（更多抑郁症状/副作用） |
| 三环类抗抑郁药单药/辅助 | 2阴性（躁狂转相风险） |

### 3.4 维持治疗临床选择指南

| 药物 | 最佳适用特征 |
|------|------------|
| **锂盐** | 经典发作-缓解病程、BD家族史（尤其锂应答家族史）、低共病率、典型临床表现 |
| **拉莫三嗪** | 主要抑郁极性 + 共病焦虑；**不适合频繁躁狂发作者** |
| **喹硫平** | 混合特征（同时预防躁狂和抑郁及混合） |
| **阿塞那平** | 同时预防躁狂和抑郁 |
| **卡马西平** | 非典型病程、BDII或分裂情感障碍 |

> **锂盐应答预测因子**：发作-缓解病程、BD家族史+锂应答、低共病（尤其焦虑/物质滥用）、躁狂-抑郁-缓解双相发作模式、典型表现、锂应答家族史（67% vs 35%）、正常脑电图、高脑锂浓度、MRS可见NAA升高/mI降低。

## 四、双相II型障碍（Section 6）

### 4.1 概述

- 患病率0.67%（BDI为0.87%）
- 致残性 ≈ BDI，经济负担可达BDI的4倍
- 1/3患者终生自杀未遂，1/25自杀身亡
- 躁狂期抑郁占比更大

### 4.2 双相II型急性轻躁狂

- 多数标准抗躁狂药未在轻躁狂中研究
- 临床经验：所有抗躁狂药对轻躁狂有效
- 推荐锂盐/丙戊酸 ± 非典型抗精神病药
- 停用抗抑郁药和兴奋剂

### 4.3 双相II型抑郁

## 五、治疗依从性（Section 5.2）

### 依从性差的风险因素

| 类别 | 风险因素 |
|------|---------|
| 社会人口学 | 男性、年轻、低教育、单身 |
| 心理因素 | 自知力差、对治疗负面态度、副作用恐惧 |
| 共病 | 酒精/大麻使用、OCD |
| 社会因素 | 无社会活动、工作受损 |
| 病程特征 | 早发、近期住院/自杀未遂、混合发作、快速循环、精神病性症状、发作次数多 |
| 治疗因素 | 副作用、疗效不足、使用抗抑郁药、剂量不足 |

> 不依从/停药后果：50-90%停锂者3-5月复发；住院、自杀、生产力损失风险增加。
>
> 依从性干预可使依从率提高一倍以上。以患者为中心的简短心理教育最有效。
>
> 不依从者：先心理教育 → 无效则长效注射（利培酮双周LAI或阿立哌唑月剂）。

## 六、难治性双相障碍

- 排除不依从、剂量未优化、共病干扰
- 足量一线/二线药物充分疗程试用
- 考虑CYP2D6/3A4基因型检测（排除超快代谢）
- **氯氮平辅助治疗**可减少难治患者症状和总用药量（4级）
- 长效注射：利培酮LAI或阿立哌唑LAI

## 七、CANMAT证据等级与推荐体系

| 证据等级 | 定义 |
|---------|------|
| 1 | 至少1项荟萃分析/RCT/高质量队列研究 |
| 2 | 至少1项RCT或2项高质量队列研究 |
| 3 | 非对照试验/低质量RCT |
| 4 | 专家意见/病例报告 |

> 推荐等级同时考虑：效力 + 安全性/耐受性 + 维持期疗效 + 抗抑郁疗效 + 转相风险
> 列表越靠上越优先推荐，除非患者有特定原因选择排位更低的药物。

## 八、中国精神障碍诊疗规范（2020年版）—— 双相障碍

> 以下内容摘自《精神障碍诊疗规范（2020年版）》第149-153页，为中国国家卫健委发布的中国临床实践指南。与CANMAT 2018形成互补参考。
>
> **治疗阶段概述**：急性期6-8周 → 巩固治疗（抑郁4-6月/躁狂/混合2-3月）→ 维持治疗

### 8.1 双相障碍Ⅰ型急性期治疗

#### 8.1.1 躁狂发作

**药物治疗：**

| 药物类别 | 具体药物 | 推荐级别 |
|---------|---------|---------|
| 心境稳定剂 | 锂盐、丙戊酸盐 | 一线，单药或联合 |
| 第二代抗精神病药 | 喹硫平、奥氮平、阿立哌唑、帕利哌酮、利培酮、齐拉西酮、氯氮平 | 一线，单药或联合 |
| 第一代抗精神病药 | 氟哌啶醇、氯丙嗪、奋乃静 | **二线选择** |

**治疗原则：**
- 综合考虑：起效时间需求、既往单药疗效、躁狂严重程度、联合治疗安全性与耐受性、患者意愿
- 单药或联合治疗，1-2周后评估疗效及耐受性
- 拒绝服药/激越严重/口服不可靠 → 短期肌内注射抗精神病药或苯二氮䓬类

**血药浓度监测：**
- 锂盐急性期：**0.8-1.2 mmol/L**
- 丙戊酸急性期：**50-100 μg/ml**

**非药物治疗：**
- 严重躁狂、高度自杀/攻击风险、明显精神病性症状 → **ECT**
- 心理治疗及其他物理治疗证据不足

#### 8.1.2 抑郁发作

**药物治疗：**

| 药物类别 | 具体药物 | 推荐级别 |
|---------|---------|---------|
| 心境稳定剂 | 锂盐（血锂0.6-1.2 mmol/L）、拉莫三嗪、丙戊酸盐 | 单药一线 |
| 第二代抗精神病药 | 喹硫平、鲁拉西酮、奥氮平 | 单药一线 |
| 抗抑郁药辅助 | SSRI、安非他酮、阿戈美拉汀 | **加用于锂/丙戊酸/SGA充分治疗基础上** |

**抗抑郁药使用注意事项（中国规范）：**
- ⚠️ **双相障碍患者不应采用抗抑郁药单药治疗**（与CANMAT一致）
- 避免使用/慎用抗抑郁药的情况：①既往抗抑郁药诱发躁狂/轻躁狂发作史 ②混合发作或以混合状态为主要表现 ③近期出现快速循环特征
- 使用时须监测转躁及快速循环风险

**非药物治疗：**
- 难治性/快速起效需求 → ECT（严重抑郁+自杀风险、紧张症、精神病性症状、妊娠期严重抑郁）
- 联合心理干预有益：**认知行为治疗、家庭治疗、人际与社会节律治疗**（IPSRT）

#### 8.1.3 混合发作

**药物治疗：**

| 药物类别 | 具体药物 | 推荐级别 |
|---------|---------|---------|
| 抗惊厥药 | 丙戊酸盐、卡马西平 | 初始一线 |
| 心境稳定剂 | 锂盐 | 初始一线 |
| 第二代抗精神病药 | 喹硫平、奥氮平、利培酮、帕利哌酮、阿立哌唑 | 初始一线 |
| 联合治疗 | 丙戊酸盐/锂盐 + 第二代抗精神病药 | 疗效不佳时升级 |

**不推荐用于混合发作：**
- ❌ **第一代抗精神病药物**（可恶化抑郁症状）
- ❌ **抗抑郁药**（可加剧激越与易激惹）

**非药物治疗：**
- ECT有效，尤其适用于药物不佳的严重患者
- 联合心理干预有一定获益

### 8.2 双相障碍Ⅰ型巩固维持期治疗

**治疗原则：**
- 建立治疗联盟（患者+家属+医务人员），提高依从性
- 延续急性期有效方案 → 心境稳定后逐渐减量或减种类
- **不宜过快减量**，防止复燃复发
- 血药浓度辅助确定维持剂量
- 定期监测：甲状腺功能、肝肾功能、血脂、血糖、泌乳素、体重

**维持治疗药物选择：**

| 药物 | 最佳适用特征 | 备注 |
|------|------------|------|
| **锂盐** | 典型心境发作、有明显缓解期的BDI | 疗效更佳 |
| **丙戊酸盐** | 混合特征、快速循环特征 | 疗效较佳 |
| **拉莫三嗪** | 抑郁发作严重、以抑郁发作为主 | 治疗及预防抑郁效果较好 |
| 第二代抗精神病药 | 需明确心境稳定作用者 | 推荐用于维持治疗 |

**联合用药：**
- 通常疗效更佳，但联用药物剂量应小于单一用药剂量

**不推荐：**
- ❌ **第一代抗精神病药物长期用于维持治疗**
- ❌ **抗抑郁药用于BDI维持期**（即使急性期已使用，维持期也应尽量停用）
- 使用FGA诱发抑郁症状 → 尽快停用

**非药物治疗：**
- ECT可用于维持治疗，但**注意不要合并抗惊厥药**
- 心理治疗：正念治疗、IPSRT、认知行为治疗

### 8.3 双相障碍Ⅱ型急性期治疗

**治疗原则：**
- 治疗目标：尽快控制症状、缩短发作时间、争取完全/基本缓解
- 治疗期一般6-8周
- 循证证据相对BDI不足

#### 8.3.1 轻躁狂发作

**药物治疗：**

| 药物 | 备注 |
|------|------|
| 锂盐、丙戊酸盐 | 一线 |
| 喹硫平、利培酮、齐拉西酮 | 可用于抑郁和具混合特征的轻躁狂 |
| 第一代抗精神病药 | ❌ **不建议使用** |

**重要：** 必须停用使症状恶化的药物（中枢兴奋剂、抗抑郁药等）

#### 8.3.2 抑郁发作

**药物治疗阶梯：**

| 阶梯 | 药物/方案 | 备注 |
|------|---------|------|
| **一线** | **喹硫平** | 唯一一线推荐（与CANMAT一致） |
| **二线** | 锂盐（血锂0.6-1.2 mmol/L）、拉莫三嗪 | — |
| **二线** | SSRI等抗抑郁药 | **仅用于不伴混合特征的抑郁相** |
| **三线** | 丙戊酸盐、氟西汀（抑郁相）、齐拉西酮（抑郁+混合轻躁狂） | — |
| **辅助** | 阿戈美拉汀、安非他酮、甲状腺素增效 | 加用 |

**联合治疗选择：**
- 锂盐 + 丙戊酸盐
- 锂盐/丙戊酸盐 + SSRI
- 第二代抗精神病药（喹硫平、奥氮平等）+ 抗抑郁药
- 喹硫平 + 拉莫三嗪

**抗抑郁药使用原则（BDII抑郁，中国规范）：**
- 可用于：心境稳定剂单用无效、抑郁严重、抑郁持续时间长、既往治疗提示有效
- ⚠️ 原则上慎用，如使用**必须与心境稳定剂联合**
- ❌ **不推荐帕罗西汀**（安慰剂对照研究阴性）

**非药物治疗：**
- ECT：二线方案，难治/快速起效需求时可用
- 心理治疗：CBT、IPSRT、正念治疗
- 辅助：光疗、针灸、瑜伽、锻炼

### 8.4 双相障碍Ⅱ型巩固维持期治疗

**治疗原则：**
- 参考BDI，遵循全面评估与量化监测、综合治疗、积极处理共病、患者共同参与
- **重点在于预防抑郁发作**（BDII抑郁发作占比远高于轻躁狂）

**药物治疗：**

| 阶梯 | 药物 | 备注 |
|------|------|------|
| **一线** | **喹硫平、锂盐、拉莫三嗪**（单药） | — |
| 锂盐维持浓度 | 与急性期相同水平 | — |

> 注意：需定期评估利弊风险，平衡疗效和安全性；使用影响代谢的药物时应给予饮食控制和体育锻炼。

**心理治疗辅助：**
- 可提高药物治疗依从性

### 8.5 中国规范与CANMAT 2018的关键异同

| 要点 | CANMAT 2018 | 中国规范2020 |
|------|------------|-------------|
| 急性躁狂一线 | 8种SGA+锂+丙戊酸（分级证据） | 心境稳定剂+SGA（一线），FGA二线 |
| 躁狂血锂浓度 | 0.8-1.2 mEq/L | 0.8-1.2 mmol/L |
| BDI抑郁一线 | 喹硫平(1)、锂(2)、拉莫三嗪(2)、鲁拉西酮(2) | 锂盐、拉莫三嗪、喹硫平、鲁拉西酮、奥氮平 |
| BDI抗抑郁药 | 辅助SSRI/安非他酮为二线，禁单药 | 辅助SSRI/安非他酮/阿戈美拉汀，禁单药 |
| 混合发作 | 避免抗抑郁药，非典型抗精神病药±丙戊酸 | 同CANMAT；明确不推荐FGA和抗抑郁药 |
| BDI维持一线 | 锂(1)、喹硫平(1)、丙戊酸(1)、拉莫三嗪(1) | 锂盐、丙戊酸盐、拉莫三嗪、卡马西平/奥卡西平+SGA |
| BDI维持FGA | 不推荐（哌泊清2-） | 不推荐长期使用 |
| BDI维持抗抑郁药 | 三环类不推荐 | 不推荐，即使急性期已用也应停用 |
| BDII抑郁一线 | 喹硫平(1)唯一一线 | 喹硫平唯一一线 |
| BDII抗抑郁药 | 风险-获益比更优（二线SSRI/安非他酮/文拉法辛） | SSRI可用于纯抑郁相（不伴混合），不推荐帕罗西汀 |
| BDII维持一线 | 喹硫平(1)、锂(2)、拉莫三嗪(2) | 喹硫平、锂盐、拉莫三嗪 |
| ECT适应证 | 难治、快速应答需求 | 同CANMAT；BDI维持ECT需注意不合并抗惊厥药 |
| 心理治疗 | IPSRT、CBT、家庭聚焦 | CBT、家庭治疗、IPSRT；维持期加正念治疗 |
