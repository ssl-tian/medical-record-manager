# 第一部分：中国指南

## 一、抗精神病药物治疗原则

1. 确诊后尽早开始抗精神病药物治疗
2. 推荐**第二代（非典型）抗精神病药**作为一线，第一代及氯氮平作为**二线**
3. **单一用药**为原则，治疗个体化，从小剂量起始逐渐加量
4. 急性发作：继续原有效药物，无效者换药或考虑氯氮平（需监测血常规）
5. 定期评价疗效和不良反应，并对症处理
6. 药物不良反应既影响选药也影响患者是否停药，应高度重视

#### 2.1.1 氯氮平 (clozapine)

- **代谢**：口服2h达峰，生物利用度27%~47%，半衰期12h，蛋白结合率94%
- **浓度监测**：>350 ng/ml更可能有效；<250 ng/ml治疗6周无效时应调整剂量
- **代谢酶**：CYP1A2为主
- **药理特点**：低D2/高5-HT2比例，对中脑边缘系统选择性高，5-HT3阻断，强α1/α2阻断，H1/M1亲和性高，σ受体亲和性低
- **治疗剂量**：200~600 mg/d
- **适应证**：①难治性精神分裂症；②严重迟发性运动障碍；③易发生EPS者；④分裂情感障碍、难治性躁狂；⑤小剂量（25~75 mg/d）治疗继发于抗帕金森病药物的精神症状；⑥严重自杀患者
- **严重不良反应**：粒细胞缺乏症（发生率为其他药物10倍）、癫痫发作（剂量相关）
- **常见不良反应**：过度镇静、流涎、心血管影响（心率过速）、体重增加、糖脂代谢障碍、2型糖尿病
- **过量中毒**：口服>2g死亡率高达12%，主要死因心肺衰竭
- **药物相互作用**：氟伏沙明（强CYP1A2抑制剂）可增高浓度；苯妥英/卡马西平诱导CYP2C19/3A4可降低浓度（≤50%）；利培酮弱CYP2D6抑制剂可升高浓度
- **撤药症状**：突然停药常出现胆碱能反跳、精神症状恶化、严重运动障碍，应在严格适应证下逐渐停用

#### 2.1.2 利培酮 (risperidone)

- **代谢**：口服1h达峰，生物利用度70%~82%，CYP2D6代谢为9-羟利培酮
- **半衰期**：母药3h，9-羟利培酮24h
- **药理特点**：5-HT2A/D2强拮抗，α1/α2高亲和，M受体和β受体低亲和
- **治疗剂量**：2~6 mg/d
- **常见不良反应**：剂量相关EPS、血催乳素水平增高、镇静、头晕
- **药物相互作用**：氟西汀/帕罗西汀（CYP2D6抑制）可阻断代谢；卡马西平（酶诱导剂）增强代谢

#### 2.1.3 奥氮平 (olanzapine)

- **代谢**：口服5h达峰，半衰期31h（21~54h），每日1次用药，CYP1A2/CYP2D6代谢
- **蛋白结合率**：93%
- **药理特点**：5-HT2A/D2/D1/D4阻断，M1/H1/5-HT3阻断，5-HT2A阻断约为D2的8倍
- **治疗剂量**：5~20 mg/d，高剂量可达30 mg/d
- **常见不良反应**：短暂镇静、体位性低血压、**体重增加明显**、EPS危险较低、恶性综合征、暂时性催乳素升高
- **药物相互作用**：卡马西平/苯妥英诱导CYP3A中度降低浓度（≤50%）；西米替丁可能增高浓度

#### 2.1.4 喹硫平 (quetiapine)

- **代谢**：口服1~1.5h达峰，半衰期6.9h，48h达稳态
- **蛋白结合率**：83%，95%以上以代谢产物排泄
- **代谢酶**：CYP3A4主要途径
- **药理特点**：5-HT2/H1/5-HT6/α1高亲和，D2中度亲和，D1/M1/D4极低亲和
- **治疗剂量**：300~750 mg/d
- **不良反应特点**：**基本不引起EPS**，**不引起泌乳素升高**（或仅暂时性）；嗜睡、头晕、体位性低血压；偶尔QTc延长；甲状腺激素轻度降低（无临床意义）
- **药物相互作用**：苯妥英（CYP3A4诱导）增加清除率5倍；硫利达嗪增加清除率60%

#### 2.1.5 齐拉西酮 (ziprasidone)

- **代谢**：口服6~8h达峰，**必须与食物同服**（生物利用度增加1倍至100%），半衰期6~10h
- **蛋白结合率**：>99%
- **药理特点**：5-HT2A/D2比值11:1，D3强亲和，5-HT1A强激动，NE再摄取中度抑制
- **治疗剂量**：急性期80~160 mg/d分两次与食物同用；维持期40~160 mg/d
- **注射剂**：10~20 mg/次，最高40 mg/d，疗程≤3天
- **不良反应特点**：**体重增加轻微**，**不影响糖脂代谢**；高剂量有剂量相关EPS；QTc延长风险
- **心电图要求**：必须进行基线心电图检查，并定期检查
- **药物相互作用**：对CYP2D6/2C9/2C19/3A4/1A2抑制作用很弱

#### 2.1.6 阿立哌唑 (aripiprazole)

- **代谢**：口服3~5h达峰，生物利用度87%，半衰期75h
- **代谢酶**：CYP3A4、CYP2D6多重途径
- **药理特点**：**5-HT-DA系统稳定剂**，突触后D2弱激动（过高下调DA，过低上调DA），5-HT2A阻断，5-HT1A部分激动
- **治疗剂量**：10~30 mg/d，起始10~15 mg/d，>30 mg/d提高剂量疗效并不增加
- **不良反应特点**：**EPS无明显增加**，**泌乳素轻度下降或不升**，**脂代谢影响不显著**；静坐不能（与剂量无明显关系）
- **药物相互作用**：CYP2D6/3A4抑制剂可提高血药浓度

#### 2.1.7 氨磺必利 (amisulpride)

- **药理特点**：选择性D2/D3受体结合，不与5-HT受体、胆碱能、肾上腺素能受体结合
  - **低剂量**（≤300 mg/d）：阻断突触前D2/D3受体 → 改善**阴性症状**
  - **高剂量**（>300 mg/d）：阻断边缘系统中部多巴胺神经元 → 治疗**阳性症状**
- **治疗剂量**：急性发作400~800 mg/d，可提高至1200 mg/d
- **不良反应**：>300 mg/d易致EPS；**泌乳素升高常见**（乳溢、闭经、男子乳腺发育、阳痿）
- **排泄**：经肾脏排泄，肾功能不全需减量（Ccr 30~60减半，Ccr 10~30减至1/3）
- **药物相互作用**：肝代谢量很小，预期不与CYP450发生显著相互作用
- **禁忌联用**：Ia类/III类抗心律失常药、某些精神镇静药物

#### 2.1.8 帕利哌酮 (paliperidone)

- **药理特点**：利培酮活性代谢物（9-羟利培酮），D2/5-HT2A阻断，**D2快解离**→降低EPS和高泌乳素血症
- **代谢**：不经CYP450代谢，59%以原型从尿液排出
- **治疗剂量**：3~12 mg/d，推荐起始6 mg/d，**无需滴定**
- **常见不良反应**：静坐不能、EPS、**高泌乳素血症较常见**

#### 各药物要点

**氯丙嗪**：低效价，治疗剂量偏高（200~600 mg/d），多受体作用，兼有强镇静作用，主要不良反应：过度镇静、抗胆碱能样作用、心血管反应、致痉挛作用。半衰期8~35h。

**奋乃静**：高效价D2拮抗剂，CYP2D6代谢，半衰期8~21h，治疗剂量20~60 mg/d，主要不良反应为EPS，对躯体器官系统影响较小。

**氟哌啶醇**：高效价，D2受体选择性最强的阻断剂，半衰期15~25h，有效治疗剂量6~20 mg/d，维持量2~6 mg/d。主要不良反应为EPS，可引发心脏传导阻滞（有猝死报告）。

**舒必利**：苯甲酰胺类，选择性D2阻断，主要作用于边缘系统，EPS较其他典型药物略低。低剂量200~600 mg/d有一定抗焦虑抑郁作用，>1000 mg/d治疗阳性症状。静脉滴注200~600 mg/d连续1~2周对紧张症有效。半衰期约8h。主要不良反应：泌乳素升高、失眠、烦躁。

#### 第一代药物的局限性

1. 不能改善认知功能（抗胆碱能作用可能使记忆恶化）
2. 对原发阴性症状疗效微小，可产生继发性阴性症状与抑郁
3. 约30%患者阳性症状不能有效缓解
4. EPS和TD比例较高
5. 患者依从性不好
6. 对社会功能和自我照料能力改善作用较小

## 三、急性期药物治疗程序

### 3.1 治疗方案决定前评估

1. 靶症状潜在原因全面评估（体格检查、精神检查、环境因素）
2. 一般医疗条件和医学资源
3. 代谢障碍和心血管疾病风险家族史
4. 以往用药过程和结果（剂量、时间、疗效、不良反应、依从性）

### 3.2 首发精神分裂症

- **单一用药**为原则，起始剂量用标准剂量范围下限
- **优先口服**，非口服应尽量缩短时间（新型注射剂如齐拉西酮肌注限于3天内）
- **一线选择**（SGAs）：阿立哌唑、氨磺必利、奥氮平、喹硫平、帕利哌酮、利培酮、齐拉西酮
- **不推荐氯氮平作为首发一线选择**
- FGA对首发患者也有效，但EPS易感性高，靶症状局限于阳性症状群
- 不应突然停药（否则症状反跳可误认为复发）

### 3.3 急性复发/恶化

- 回顾既往方案（药物选择、剂量、持续时间、依从性、中断原因、不良反应、共病）
- 在起始剂量、目标剂量、滴定时间等方面可采取**相对积极**的策略
- 对足量足疗程（≥2种不同类别）仍无效者考虑氯氮平

### 3.4 按症状群的处理流程

#### 3.4.1 幻觉妄想为主

**不合作患者**：
1. 氟哌啶醇/氯丙嗪肌注，或齐拉西酮肌注（20~40 mg/d，连续≤3天）→ 口服序贯
2. 可合并BZD肌注（劳拉西泮/氯硝西泮）
3. 可给予口服液/口崩片：利培酮、帕利哌酮、奥氮平、喹硫平、齐拉西酮、阿立哌唑、氨磺必利

**合作患者**：
1. 首选口服非典型药（利培酮/帕利哌酮/奥氮平/喹硫平/齐拉西酮/阿立哌唑/氨磺必利），其次典型药
2. 3天~2周滴定至目标剂量，持续治疗6~8周评估
3. 无效 → 换药/长效制剂/氯氮平/联合ECT
4. 再无效 → 联合治疗/增效/ECT

#### 3.4.2 兴奋激越为主

1. 氟哌啶醇/齐拉西酮肌注 ± BZD肌注 → 口服序贯
2. 或口服非典型药 + 注射BZD
3. **避免首选氯氮平**
4. 无效 → 氯氮平/丙戊酸钠/ECT

#### 3.4.3 紧张症群

1. **首选ECT或舒必利静滴**（起始50~100 mg/d，3~5天滴定至200~600 mg/d，持续1~2周）
2. 治疗有效后口服舒必利或转换非典型药
3. 注意躯体营养及水电解质平衡

#### 3.4.4 阴性症状为主

首选非典型药（目标剂量相对较低）：

| 药物 | 推荐剂量范围(mg/d) |
|------|-------------------|
| 氨磺必利 | 200~300 |
| 阿立哌唑 | 10~20 |
| 利培酮 | 2~4 |
| 帕利哌酮 | 3~6 |
| 奥氮平 | 5~10 |
| 喹硫平 | 100~300 |
| 齐拉西酮 | 40~80 |

无效 → 换药/氯氮平（50~100 mg/d）→ 联合治疗

#### 3.4.5 伴抑郁心境症状

1. 首选非典型药（阿立哌唑/氨磺必利/喹硫平/齐拉西酮/利培酮/帕利哌酮/奥氮平）或舒必利或小剂量氯氮平
2. 无效 → 换另一种非典型药
3. 仍无效 → **联合抗抑郁药**（SSRI/SNRI/NaSSA/TCA）
4. 严重消极/自杀/拒食 → 联合ECT

#### 3.4.6 自杀/自伤突出

1. **首选**：奥氮平、阿立哌唑、氨磺必利、齐拉西酮、帕利哌酮 ± ECT
2. 无效 → 评估自杀风险与抑郁相关性：
   - 抑郁明显 → 联合新型抗抑郁药（SSRI/SNRI/米氮平）
   - 与精神病性症状相关 → **换用氯氮平**（循证支持最多的选择）
3. 仍无效 → 氯氮平联合ECT

## 四、慢性精神分裂症急性恶化

### 治疗原则

1. FGA和SGA均能有效治疗急性恶化/复发
2. 选药必须个体化，取决于既往用药和不良反应体验
3. 所有不良反应均应重视，特别关注EPS/代谢/心血管不良反应
4. **换药前最佳剂量治疗至少2周但不超过8周**
5. **不依从是复发主要原因**
6. 快速剂量递增、高负荷剂量、超说明书高剂量治疗**不推荐**

## 五、难治性精神分裂症（TRS）

### 定义（Conley & Kelly修改版）

过去5年经过至少**2种足量（≥600 mg/d CPZ等效）**抗精神病药治疗**4~6周**后未获临床改善：
- BPRS总分 ≥45分
- CGI-S ≥4分
- 或者4项阳性症状中至少2项 ≥4分

发生率：起病初期约10%~15%为TRS，最终约30%~60%发展为TRS。

### 治疗策略

#### 5.1 换为氯氮平（首选）

- **足量**：300~800 mg/d
- **足疗程**：至少8周
- 疗效确切，还可改善自杀风险和攻击性行为
- 需密切监测血液系统和代谢不良反应

#### 5.2 氯氮平联合增效剂（表7-6）

| 治疗策略 | 证据分类 | 推荐 |
|---------|---------|------|
| 氯氮平+拉莫三嗪 | B | 3 |
| 氯氮平+西酞普兰 | B | 3 |
| 氯氮平+ECT | B | 3 |
| 氯氮平+抗精神病药 | B | 3 |
| 雌激素 | B | 3 |
| 经颅直接电流刺激 | B | 3 |
| 氯氮平+托吡酯 | D | 5 |
| 氯氮平+氟西汀 | D | - |
| 氯氮平+米氮平 | D | - |
| rTMS | D | 5 |

> 30%~50% TRS患者对氯氮平仍无反应或不耐受，此时考虑联合增效策略，但证据总体很弱。

#### 5.3 换为其他第二代药

- 利培酮（6项RCT）：安全性和耐受性优于氯氮平，总体疗效不优于氯氮平
- 奥氮平（4项研究）：总体疗效不如氯氮平，但不良反应少于FGA
- 喹硫平：600→1200 mg/d耐受性好但疗效相当；个案报告高剂量（1200~2400 mg/d）可能改善阳性症状/攻击行为
- 阿立哌唑：氯氮平无效/不耐受者换为阿立哌唑可获改善
- 齐拉西酮：疗效介于氯氮平和氯丙嗪之间，不良反应较少

#### 5.4 ECT

短期治疗伴或不伴抗精神病药，可获得总体显著改善。对心血管和认知功能的不良反应较常见。

#### 5.5 其他

rTMS等证据不多，仅有个案报告显示可能短期有效。

## 六、长期治疗

- 所有患者需要维持治疗
- 稳定期重点：症状缓解、功能改善、复发预防
- 药物选择必须个体化
- 心理社会干预作为辅助
- 不依从是复发的主要原因，依从性尤为重要

## 八、首发精神分裂症（First-episode psychosis）

### 8.1 定义与鉴别

首发精神病（first-episode psychosis）指患者首次出现显著行为异常，包括幻觉、妄想、激越、木僵、意识混浊和紧张症症状。临床上需与以下疾病鉴别：

| 疾病 | 症状持续时间 |
|------|-------------|
| 短暂性精神病性障碍 | 1天~1个月 |
| 分裂样障碍 | 1~6个月 |
| 精神分裂症 | >6个月 |
| 分裂情感性障碍 | >6个月 |

> 核心原则：首先确保将精神分裂症与表现为类似症状的各种躯体疾病鉴别开来。

### 8.2 首发治疗推荐（JSCNP CQ1-1）

**推荐等级与要点：**

| 推荐 | 证据等级 |
|------|---------|
| SGA优于FGA用于首发精神病治疗 | **2A** |
| 选择SGA时，不推荐特定药物优先 | 2D |
| 对治疗效应和不良反应高度敏感，低剂量起始逐渐加量 | **共识** |
| SGA之间不做明确优劣排名 | **共识** |

**具体药物证据：**

| 药物 | 首发有效剂量范围 | 证据 |
|------|----------------|------|
| 阿立哌唑 | 9.9~20.0 mg/d（短期）；11.6~14.5 mg/d（长期） | D |
| 奥氮平 | 8.7~17.0 mg/d（短期）；8.7~12.6 mg/d（长期） | C |
| 帕利哌酮 | 6.4 mg/d（长期，但需注意血脂异常） | D |
| 喹硫平 | 358~506 mg/d | C |
| 利培酮 | 2~4 mg/d 低剂量即可获得高效 | C |
| 氟哌啶醇 | 2 mg/d 低剂量即可，8 mg/d 不优于2 mg/d但EPS更多 | C |

**SGA各药物在首发患者中的比较证据：**

| 对比 | 结果 |
|------|------|
| 阿立哌唑 vs 喹硫平（52周RCT） | 阿立哌唑全因停药率显著更低，疗效/EPS/体重/泌乳素无显著差异 |
| 阿立哌唑 vs 帕利哌酮（52周RCT） | 帕利哌酮疗效优于阿立哌唑，但帕利哌酮降低HDL、升高甘油三酯 |
| 奥氮平 vs 利培酮（RCT） | 疗效无差异；奥氮平体重增加更明显，利培酮EPS更多 |
| 奥氮平 vs 氟哌啶醇（52周RCT） | 奥氮平因疗效不足停药率显著更低，但体重增加显著 |
| 喹硫平 vs 氟哌啶醇（52周RCT） | 疗效无差异，两组停药率相似 |

### 8.3 首发患者需关注的四大不良反应

1. **EPS**（含静坐不能、肌张力障碍、运动障碍）
2. **代谢综合征**（体重增加、血脂异常、高血糖）
3. **内分泌异常**（如高泌乳素血症）
4. **心血管异常**（如QTc延长）

### 8.4 治疗效应判定时间（JSCNP CQ1-3）

| 推荐 | 证据等级 |
|------|---------|
| ~60%~70%患者可能在用药2~4周内出现治疗应答 | **2D** |
| 用药2周内的早期应答可预测12周后的治疗反应 | D |
| 但部分患者治疗应答可能需要更长时间 | **共识** |
| 建议至少等待**2~4周**判定治疗反应 | **2D** |
| 若低剂量下2~4周反应不足，可考虑提前加量 | 2D |

**关键证据：**
- Emsley等（n=522）：35.6%和59.4%分别在2周和4周时达到治疗应答标准
- 6周时应答可预测后续缓解；6周未应答者仍可能达到后续缓解标准
- 尚无基于2~4周治疗反应决定换药/继续用药的RCT

### 8.5 首发治疗持续时间（JSCNP CQ1-4）

| 推荐 | 证据等级 |
|------|---------|
| 抗精神病药持续用药可降低复发率，至少1年 | **1A** |
| 症状缓解后持续用药至少**1年**以预防复发 | **1A** |
| 减量/停药后复发率约为持续用药组的**2倍** | A |
| 7年随访：减量/停药组恢复率显著高于持续用药组（约2倍） | C |
| 减药决策应充分告知风险与获益后与患者共同决定 | 共识 |

> Wunderink等7年随访研究是首个高质量证据提示减量/停药长期获益的报告，但减量组多数为仅减量而非完全停药，表明病情允许减量的患者预后更佳。

## 九、复发与恶化（Recurrence and relapse）

### 9.1 定义

- **复发（recurrence）** / **恶化（relapse）**：症状缓解或部分缓解后3~6个月，评估量表（PANSS/BPRS）评分恶化
- 复发的主要原因：抗精神病药不依从、重大生活事件（应激），但也可是精神分裂症自然病程

### 9.2 复发处理总原则（JSCNP Chapter 2 总结）

| 原则 | 说明 |
|------|------|
| 1. 先确认用药是否合理 | 确认当前剂量、用药时间、依从性是否适当，再考虑换药或加量 |
| 2. 因停药复发者 | 选择重新起始的抗精神病药时，应考虑既往用药的疗效和不良反应 |
| 3. 依从性好但复发 | 如可加量，推荐先增加剂量 |
| 4. 加量无效再换药 | 剂量加至可耐受范围和推荐剂量后观察2~4周，8周仍无反应则换药 |
| 5. 推荐单药治疗 | 联合用药效果不确定，且可能增加不良反应 |

### 9.3 加量 vs 换药（JSCNP CQ2-1）

| 推荐 | 证据等级 |
|------|---------|
| 先确认当前剂量、用药时间、依从性是否适当 | **1D** |
| 停药复发者，应参考既往用药反应（含不良反应）选择重启药物 | **1D** |
| 依从性好且血药浓度有效但反应差 → 推荐换药 | 2D |
| 仍有加量空间且耐受好 → 推荐先加量 | 2D |
| 加量观察2~4周，最迟8周无反应则换药 | 2C |
| 优先选择可测血药浓度的药物（如氟哌啶醇）或LAI以排除假性耐药 | 2D |
| 快速加量或超推荐剂量证据不足，不建议 | 2D |
| **总体推荐：复发时优先尝试加量而非换药** | **2D** |

**关键证据：**
- 用药2周内改善幅度大于后续任何时期
- 2周内症状改善<20%~25%者，后续改善概率约20%（预测准确率~80%）
- 2~6周观察期的反应可准确反映后续反应和缓解
- 尚无>8周观察期的报告

### 9.4 复发时各药物证据与推荐剂量（JSCNP CQ2-2）

#### 安慰剂对照RCT证据总结

| 药物 | 有效剂量 | 效力 | 耐受性 | 主要注意事项 |
|------|---------|------|--------|-------------|
| 阿立哌唑 | ≥10 mg/d | **A** | **A** | ≥10mg/d才显示效力 |
| 布南色林 | 2.5~10 mg/d | **B** | — | 10mg/d效力更高但EPS增加 |
| 氟哌啶醇 | ≥10 mg/d（A）；≥4 mg/d（B） | **A** | — | EPS发生率高（4mg/d即有EPS） |
| 奥氮平 | ≥10 mg/d | C | — | 体重增加需警惕（A） |
| 喹硫平 | ≥250 mg/d（B）；可能≥150 mg/d（C） | B~C | **A** | 效力证据弱~中等但耐受性好 |
| 利培酮 | ≥2 mg/d | **A** | — | 泌乳素升高（A）、药源性帕金森综合征（B）常见 |
| 佐替平 | ≥150 mg/d | C | — | — |
| 氯丙嗪 | 1000 mg/d | B | — | 样本量小，可靠性有限 |
| 氟奋乃静 | — | — | — | 仅有1971年1篇旧报告 |

#### SGA vs FGA 头对头比较（10项RCT，对照药均为氟哌啶醇）

| 比较 | 结果 | 证据等级 |
|------|------|---------|
| SGA vs FGA（总体） | 疗效相似；SGA的EPS更少、泌乳素升高更小 | **A** |
| 阿立哌唑 vs 利培酮 | 疗效相似；利培酮泌乳素更高、EPS更多（C） | A |
| 利培酮 vs 齐拉西酮 | — | — |

> **结论：SGA整体优于FGA（复发/恶化治疗）**，主要优势在于安全性。

#### 日本临床试验新药（截至2014年12月）

阿塞那平（asenapine）、卡利拉嗪（cariprazine）、鲁拉西酮（lurasidone）、齐拉西酮（ziprasidone）——均已确认对复发/恶化有效，但当时尚未在日本获批。

### 9.5 抗精神病药联合治疗（JSCNP CQ2-3）

| 推荐 | 证据等级 |
|------|---------|
| 联合治疗可能比单药更有效，但效果不确定且可能增加不良反应 | **C** |
| **复发/恶化时不推荐常规联合用药** | **2C** |
| 仅在单药治疗（包括氯氮平）反应不佳的严重病例中谨慎使用 | 2C |

**正方理由：** 更快速强大的效应、改善各种症状（激越、认知损害、阴性症状）、改善共病症状（失眠、焦虑、抑郁）

**反方风险：** 总剂量增加、急性/迟发性不良反应增加、不可预测的药物相互作用、难以识别致病药物、依从性下降、死亡率增加、医疗成本增加

**已有证据的特定联合方案：**
| 联合方案 | 结果 |
|---------|------|
| 奥氮平+利培酮 | 对精神症状可能优于单药 |
| 利培酮/喹硫平+阿立哌唑 | 无效 |
| 氯氮平+阿立哌唑 | 改善阴性症状 |
| 氯氮平+利培酮 | 改善高泌乳素血症 |
| 氯氮平+氟西汀 | 改善氯氮平所致体重增加 |

> 短期联合（<3周）：显示改善效应；长期联合：无优势且阴性症状可能恶化。

### 9.6 抗精神病药合并其他精神药物（JSCNP CQ2-4）

| 药物类别 | 推荐 | 证据等级 |
|---------|------|---------|
| **苯二氮䓬类（BZ）** | 短期有效（72小时内改善激越），但不推荐长期使用（依赖性、可能增加死亡率） | 2D |
| **丙戊酸** | <3周短期联合有效（D）；>3周长期联合阴性症状可能恶化（C）；不推荐长期使用 | 2D |
| **卡马西平** | 系统综述显示阴性结果，不推荐 | 2D |
| **锂盐** | 无RCT显示在复发/恶化时有效 | — |
| **抗抑郁药** | 证据不明确，1项小样本研究（n=12）显示奥氮平+氟伏沙明可能改善症状，但可能是氟伏沙明升高了奥氮平血药浓度 | 2D |

> **结论：复发/恶化时不推荐常规合并其他精神药物。**
