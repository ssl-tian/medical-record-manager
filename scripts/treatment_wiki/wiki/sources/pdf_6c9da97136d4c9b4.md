# 中国双相障碍防治指南(2025版)——药物治疗章节(第4章)

## 摘要

《中国双相障碍防治指南(2025版)》第4章系统阐述了双相障碍的全病程药物治疗。强调9大治疗原则（充分评估量化监测、综合治疗、全病程、全面治疗、提高依从性、优先住院/MECT、患方共同参与、治疗共病、生活方式管理）。全病程分2期：急性治疗期（6-8周）→巩固维持期（数年）。躁狂急性期一线单药8种（锂盐/喹硫平/丙戊酸盐/阿塞那平/阿立哌唑/帕利哌酮>6mg/利培酮/卡利拉嗪），联合推荐喹硫平/阿立哌唑/利培酮/阿塞那平+锂盐/丙戊酸盐。双相抑郁一线单药6种（喹硫平/鲁拉西酮/卡利拉嗪/锂盐/拉莫三嗪/丙戊酸盐），一线联合奥氟合剂及鲁拉西酮+锂盐/丙戊酸盐。抗抑郁药不推荐单用，必须联用心境稳定剂或非典型抗精神病药。维持期一线：锂盐/拉莫三嗪/利培酮LAI/奥氮平/喹硫平，联合推荐喹硫平+锂盐/丙戊酸盐。利培酮和阿立哌唑长效针剂均有维持期证据。

## 识别的实体

- [[锂盐]] (drug) — 置信度: EXTRACTED
  > 经典心境稳定剂；躁狂急性期一线单药(1A)、抑郁急性期一线(1B)、维持期一线(1B)；治疗剂量和中毒剂量接近，需定期监测血锂浓度；对自杀有独立预防作用
- [[丙戊酸盐]] (drug) — 置信度: EXTRACTED
  > 心境稳定剂/抗惊厥药；躁狂急性期一线单药(1A)、抑郁急性期一线(1B)；与拉莫三嗪联用增加严重皮疹风险；与锂盐联用增加震颤/体重增加/胃肠道不适
- [[拉莫三嗪]] (drug) — 置信度: EXTRACTED
  > 心境稳定剂/抗惊厥药；抑郁急性期一线单药(1A)、维持期一线单药(1B)；对躁狂急性期无效(不推荐/A)；短期和长期安全性好
- [[卡马西平]] (drug) — 置信度: EXTRACTED
  > 心境稳定剂/抗惊厥药；躁狂急性期二线单药(2A)、抑郁急性期二线(2C)；酶诱导剂，可能降低联用药物的血药浓度
- [[奥卡西平]] (drug) — 置信度: EXTRACTED
  > 卡马西平类似物；躁狂三线联合治疗(3C)；维持期三线联合锂盐(3C)
- [[喹硫平]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期一线单药(1A)；抑郁急性期一线单药(1A)；维持期一线单药(1B)；联合锂盐/丙戊酸盐用于躁狂(1A)、抑郁(2B)和维持期(1B)均有推荐
- [[奥氮平]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期二线单药(2A)；抑郁急性期二线单药(2A)；维持期一线单药(1B)；因安全性/耐受性从一线降为二线（躁狂）；+锂盐/丙戊酸盐用于躁狂二线(2A)和抑郁二线(2A)
- [[阿立哌唑]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期一线单药(1A)；联合锂盐/丙戊酸盐一线(1B)；维持期二线单药(2C)；AOM长效针剂400mg/月维持期二线(2C)，可显著推迟复发时间
- [[利培酮]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期一线单药(1A)；联合锂盐/丙戊酸盐一线(1A)；维持期一线长效针剂(1B)；有口崩片剂型用于激越二线(2C)
- [[帕利哌酮]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期一线单药(>6mg,1A)；作为联合治疗的疗效证据不足；维持期三线缓释剂(3C)
- [[阿塞那平]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期一线单药(1A)；联合锂盐/丙戊酸盐一线(1B)；维持期二线单药(2C)；舌下含服剂型可用于激越二线(2C)
- [[卡利拉嗪]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期一线单药(1A)；抑郁急性期一线单药(1B)，1.5mg和3mg均可改善MADRS评分
- [[鲁拉西酮]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；抑郁急性期一线单药(1A)；联合丙戊酸盐/锂盐一线(1A)；维持期联合锂盐/丙戊酸盐二线(2C)
- [[齐拉西酮]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期二线单药(2A)；抑郁急性期三线(3C)；维持期联合锂盐/丙戊酸盐二线(2C)；IM注射用于激越二线(2C)
- [[氯氮平]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；躁狂急性期三线(3D)；仅在其他治疗均无效时考虑
- [[氟哌啶醇]] (drug) — 置信度: EXTRACTED
  > 典型抗精神病药；躁狂急性期二线单药(2A)；IM注射用于激越二线(2C)；联合咪达唑仑/异丙嗪IM用于激越二线(2C)；总体不良反应偏大，长期使用有TD风险，有诱发转抑郁相风险
- [[氯丙嗪]] (drug) — 置信度: EXTRACTED
  > 典型抗精神病药；躁狂急性期三线单药(3B)；有良好的抗躁狂和镇静作用，但不良反应偏大
- [[奥氟合剂(OFC)]] (drug) — 置信度: EXTRACTED
  > 奥氮平+氟西汀复方制剂；抑郁急性期一线联合治疗(1A)；显著减轻抑郁症状且不增加转躁风险
- [[劳拉西泮]] (drug) — 置信度: EXTRACTED
  > 苯二氮䓬类；激越症状一线IM注射(1B)
- [[氯硝西泮]] (drug) — 置信度: EXTRACTED
  > 苯二氮䓬类；躁狂急性期三线单药(3B)
- [[他莫昔芬]] (drug) — 置信度: EXTRACTED
  > 选择性雌激素受体调节剂/蛋白激酶C抑制剂；躁狂急性期三线单药(3B)；+锂盐/丙戊酸盐三线(3B)；有子宫癌风险
- [[哌罗匹隆]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药；抑郁急性期联合锂盐/丙戊酸盐二线(2B)
- [[布瑞哌唑]] (drug) — 置信度: EXTRACTED
  > 非典型抗精神病药/D2部分激动剂；抑郁三线联合SSRIs(3C)
- [[舍曲林]] (drug) — 置信度: EXTRACTED
  > SSRI抗抑郁药；抑郁三线+锂盐(3C)；不推荐单用于双相抑郁
- [[心境稳定剂]] (drug_class) — 置信度: EXTRACTED
  > 双相障碍核心治疗药物类别；主要成员为锂盐、丙戊酸盐、拉莫三嗪、卡马西平；具有抗躁狂和/或抗抑郁作用，预防复发
- [[非典型抗精神病药]] (drug_class) — 置信度: EXTRACTED
  > 在双相障碍各期治疗中均有重要地位；具有心境稳定作用，适用于维持治疗；长期使用需监测代谢综合征和泌乳素升高等不良反应
- [[全病程治疗]] (concept) — 置信度: EXTRACTED
  > 双相障碍治疗分2期：急性治疗期（6-8周，以症状达到临床缓解为标志）和巩固维持期（数年，部分患者需长期/终身治疗）
- [[快速循环]] (concept) — 置信度: EXTRACTED
  > 12个月内≥4次情感发作（躁狂/轻躁狂/抑郁/混合）；抗抑郁药可能诱发；治疗上避免使用抗抑郁药，以心境稳定剂为核心
- [[双相I型障碍]] (disease) — 置信度: EXTRACTED
  > 以至少1次躁狂发作为特征，可伴或不伴抑郁发作；本章涵盖躁狂急性期、抑郁急性期、维持期和激越处理4大临床场景
- [[双相II型障碍]] (disease) — 置信度: EXTRACTED
  > 至少1次轻躁狂发作+至少1次抑郁发作，无躁狂发作史；RCT证据较少，部分治疗证据来自CANMAT；本指南首次单独列出
- [[改良电休克治疗(MECT)]] (concept) — 置信度: EXTRACTED
  > 躁狂急性期二线(2C，有效率达80%)、抑郁急性期三线(3B，优于药物方案)；优先用于高自杀风险/紧张症/药物难治/妊娠期严重发作
- [[激越]] (concept) — 置信度: EXTRACTED
  > 精神科急性综合征，表现为坐立不安和过多肢体活动伴焦虑；躁狂发作常见伴随症状；需快速识别和处理
- [[中国双相障碍防治指南(2025版)]] (guideline) — 置信度: EXTRACTED
  > 中国双相障碍诊疗权威指南，2025年发布；第4章涵盖治疗原则、躁狂急性期、抑郁急性期、维持期、双相II型障碍的全流程药物推荐

## 知识关联

- [[锂盐]] → [[双相I型障碍]]: 唯一在躁狂急性期(1A)、抑郁急性期(1B)、维持期(1B)均获一线推荐的经典心境稳定剂
- [[喹硫平]] → [[双相I型障碍]]: 唯一在躁狂急性期(1A)、抑郁急性期(1A)、维持期(1B)均获一线推荐的非典型抗精神病药
- [[拉莫三嗪]] → [[躁狂急性期]]: 对躁狂急性期无效，不推荐用于急性躁狂治疗(A级证据)
- [[拉莫三嗪]] → [[丙戊酸盐]]: 联用增加严重皮疹风险，应特别谨慎
- [[抗抑郁药]] → [[双相抑郁]]: 不推荐单用；必须联用心境稳定剂或非典型抗精神病药；维持期缓解后8周内停用
- [[丙戊酸盐]] → [[锂盐]]: 联用增加震颤、体重增加和胃肠道不适
- [[奥氮平]] → [[代谢综合征]]: 因代谢不良反应在躁狂急性期从一线降为二线推荐
- [[MECT]] → [[躁狂急性期]]: 二线治疗(2C)，对高达80%患者有效
- [[MECT]] → [[双相抑郁]]: 三线治疗(3B)，明显优于药物方案
- [[利培酮长效针剂]] → [[维持期]]: 唯一获1B级推荐的长效针剂，预防任何情感发作相复发优于安慰剂
- [[AOM(阿立哌唑长效针剂)]] → [[维持期]]: 2C级推荐，仅对躁狂相有效，对抑郁/混合相无明显预防作用
- [[奥氟合剂(OFC)]] → [[双相抑郁]]: 一线联合治疗(1A)，不增加转躁风险
- [[卡马西平]] → [[药物相互作用]]: 酶诱导剂，可降低联用药物（奥氮平/利培酮等）的血药浓度
- [[抗抑郁药]] → [[快速循环]]: 可能诱发快速循环，禁用于快速循环/混合发作/既往转躁史患者
- [[中国双相障碍防治指南(2025版)]] → [[CANMAT/ISBD 2018]]: 均推荐锂盐为核心心境稳定剂；中国指南首次单独列出双相II型障碍治疗章节
